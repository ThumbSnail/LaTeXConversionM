Features

1. Prints out the final HTML to just copy and paste (into the "HTML" view in the Explanation inferface) all lines using LaTeX at once instead of manually entering multiple LaTeX expressions one at a time through the Explanation interface.

2. Removes italicized text by wrapping all letters/words in \text{}. If you want to keep the spaces in between multiple words like "multi word variable" then just manually place that inside of \text{}, like: \text{multi word variable}

3. Converts all * to \times

4. Simple fractions are a bit of a pain to type in LaTeX, so you can just do (whatever) / (whatever2), and it will convert it to \dfrac{whatever}{whatever2}, but you MUST wrap numerator and denominator in parentheses, even for something like (3)/(8). NOTE: This is NOT setup to handle fractions within fractions like ((3)/(5)) / ((7)/(8)). For that, you'll have to use normal LaTeX code.

(LaTeX already converts hyphens to the actual subtraction symbol, so no worries there.)
